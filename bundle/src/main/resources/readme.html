WSIT Build Tools ${project.version}

WSIT Build Tools are a set of Maven Mojo's and Ant Tasks that can be used to construct WSIT configuration files. This is particularly useful for those who do not use NetBeans to develop web services and web service clients. NetBeans has functionality built into the IDE to handle generating these WSIT configuration files, but its difficult to get this functionality in an Ant or Maven build.

These ant tasks will allow you to generate both server and client side WSIT configuration files from the WSDL of the target service.

What's New

Provides Ant tasks and Maven Mojo's to:
  • Strip all WSIT configuration from a WSDL
  • Merge any policy fragment with an existing WSDL or WSIT configuration file
  • Configure client side WSIT settings for Callback Handlers, Certificate Stores, KeyStores and TrustStores
